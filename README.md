# TypeFree

Hold a key, speak, release — your words appear right where the cursor is.

TypeFree is a macOS dictation app that works in any text field, without switching apps or breaking your flow. Built with a **Liquid Glass** UI that feels native on modern macOS.

## Download

Head to the [Releases](https://github.com/zmlabs/type-free/releases) page, grab the latest `.dmg`, and drag TypeFree into your Applications folder.

**Requirements:** macOS · Apple Silicon (M-series)

## Get Started

1. Open TypeFree and grant **Accessibility** and **Microphone** permissions when prompted
2. Pick your trigger key — `Fn` by default, or swap it for Right Command, Caps Lock, and more
3. Choose a speech recognition provider (see below)
4. Click into any text field, **hold your trigger key to record, release to insert** — or double-press while recording to cancel

## Speech Recognition Providers

| Provider | Description |
|----------|-------------|
| **Built-in** | Powered by Apple's on-device engine — no setup, no internet required |
| **OpenAI Compatible** | Works with Whisper or any OpenAI-compatible API — bring your own key |
| **Qwen3 ASR** | Alibaba's DashScope ASR — outstanding accuracy for Chinese and multilingual input |
